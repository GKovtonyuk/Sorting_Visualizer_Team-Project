# Sorting Visualizer

## 📂 Структура проєкту

```text
sorting-visualizer/
│
├── main.py              # Головний файл (візуалізація)
├── algorithms.py        # Список алгоритмів
├── sorts/               # Алгоритми сортування 
└── repost/              # Зображення проєкту та скріншоти
```
---

Для роботи проєкту потрібна бібліотека `pygame`.
---

## Як додати свій алгоритм

⚠️ **ВАЖЛИВО:** Щоб зберегти архітектуру проєкту, **не змінюйте** файли `main.py` та `algorithms.py`. Система автоматично імпортує нові файли з папки `sorts`.

###  Кроки:

1.  Зробіть **Fork** репозиторію.
2.  Клонуйте свій fork на комп’ютер:
3.  Перейдіть в папку проєкту
4.  Створіть нову гілку
5.  У папці `sorts/` створити новий `.py` файл.
6.  Реалізуйте алгоритм у форматі генератора.
###  Завершення роботи:

7.  Додайте зміни.
8.  Зробіть commit.
9.  Зробіть push.
10. Створіть **Pull Request** на GitHub.
