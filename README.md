# Sorting Visualizer

## 📂 Project Structure

```text
sorting-visualizer/
│
├── main.py              # Main file (visualization)
├── algorithms.py        # List of algorithms
├── sorts/               # Sorting algorithms
└── report/              # Project images and screenshots
```
## How to Add Your Own Algorithm

** IMPORTANT:** To preserve the project architecture, **do not modify** the `main.py` and `algorithms.py` files. The system automatically imports new files from the `sorts` folder.

### Steps:

1. Create a Fork of the repository.
2. Clone your fork to your computer.
3. Navigate to the project folder.
4. Create a new branch.
5. In the sorts/ folder, create a new .py file.
6. Implement your algorithm in generator format.

### Final Steps:

7. Add your changes.
8. Make a commit.
9. Push your changes.
10. Create a Pull Request on GitHub.

## Як додати свій алгоритм

**ВАЖЛИВО:** Щоб зберегти архітектуру проєкту, **не змінюйте** файли `main.py` та `algorithms.py`. Система автоматично імпортує нові файли з папки `sorts`.

###  Кроки:

1.  Зробіть **Fork** репозиторію.
2.  Клонуйте свій fork на комп’ютер:
3.  Перейдіть в папку проєкту
4.  Створіть нову гілку
5.  У папці `sorts/` створити новий `.py` файл.
6.  Реалізуйте алгоритм у форматі генератора.
###  Завершення роботи:

7.  Додайте зміни.
8.  Зробіть commit.
9.  Зробіть push.
10. Створіть **Pull Request** на GitHub.
